PÁGINA PRINCIPAL

Está es la página que se muestra al iniciar la aplicacion. En esta pagina se muestra un buscador.

FORMULARIO

En este formulario se añaden los dioses a la base de datos para mostrarlos en la pagina principal o a traves de el buscador. Los datos que lleva este formulario son:

  • Entrada: Nombre del dios en Español
  • Entrada: Nombre del dios en Griego
  • Entrada: Transliteracion
  • Area de texto: Significado
  • Entrada: Enlace URL de imagen relativa al dios
  • Menu de opciones: Grupo
  • Menu de opciones: Equipo
  • Area de texto: Aspecto historico
  • Area de texto: Aspecto estructural
  • Area de texto: Aspecto evolutivo

Proporciona un codigo para modificar los datos posteriormente.

PÁGINA DIOS

Esta pagina muestra el dios solicitado pidiendo los datos a la base de datos y los muestra en una tabla. Ademas proporciona dos botones para modificar o eliminar la entrada con el codigo que se creo al llenar el formulario
